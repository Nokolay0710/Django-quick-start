Django quick start
==================

- mkdir New project && cd New project

- pipenv install django==4.2.0

- pipenv shell

- django-admin startproject config .

- py manage.py runserver

- Starting development server at http://127.0.0.1:8000/

- Rocket!

- Quit the server with Ctrl+C.
